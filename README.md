# 📝 درباره پروژه

این پروژه یک سیستم پردازش تصویر و رندرینگ سه‌بعدی پیشرفته است که حرکت سر کاربر را به صورت زنده (**Real-time**) از طریق وب‌کم دریافت کرده و دقیقاً همان چرخش‌ها و زاویه‌ها را روی استخوان‌بندی (**Bone Structure**) یک مدل سه‌بعدی با فرمت `.glb` پیاده‌سازی می‌کند.

این نرم‌افزار ترکیبی هوشمندانه از فریمورک هوش مصنوعی **MediaPipe** (برای استخراج نقشه سه‌بعدی صورت) و موتور قدرتمند **Panda3D** (برای رندرینگ گرافیکی و فیلترهای PBR) است.

---

# ⚡ ویژگی‌های فنی پروژه

- **کالیبراسیون خودکار (Auto Calibration)**  
  در ۲ ثانیه اول اجرای برنامه (۶۰ فریم اول)، زاویه نشستن کاربر نسبت به وب‌کم آنالیز و صفر می‌شود.

- **فیلتر حرکتی اسموث (Motion Smoothing)**  
  جلوگیری از لرزش‌های ریز تصویر وب‌کم و شبیه‌سازی حرکت کاملاً نرم و طبیعی گردن انسان با نرخ محاسباتی داینامیک.

- **محدودیت‌های زاویه انسانی (Clamping)**  
  محدود کردن زوایای چرخش سر (`Yaw`, `Pitch`, `Roll`) برای جلوگیری از چرخش‌های غیرطبیعی و دفرمه شدن استخوان کاراکتر.

- **پشتیبانی از رندرینگ مدرن**  
  استفاده از کتابخانه `simplepbr` برای رندر متریال‌های واقع‌گرایانه (**PBR**) روی مدل‌های سه‌بعدی GLB.

- **پس‌زمینه متحرک (Dynamic GIF Background)**  
  قابلیت استخراج فریم‌های یک فایل GIF و رندر کردن آن به عنوان پس‌زمینه زنده در فضای سه‌بعدی.

# 📁 فایل‌های مورد نیاز

این مخزن شامل **تنها یک فایل اصلی پایتون** است.  
برای اجرای کامل پروژه، باید فایل‌های زیر را شخصاً تهیه کرده و دقیقاً کنار فایل پایتون قرار دهید:

1. **`destructor_head_model.glb`**  
   مدل سه‌بعدی سر یا کاراکتر شما با فرمت GLB (دارای استخوان‌بندی سر مانند `mixamorig:Head` یا `Head`)

2. **`gif.gif`**  
   فایل GIF متحرک برای بک‌گراند صحنه سه‌بعدی

# 🚀 راهنمای جامع دانلود، نصب و راه‌اندازی (قدم به قدم)

## 📥 مرحله اول: دانلود فایل پروژه از گیت‌هاب

از آنجا که این پروژه فقط شامل یک فایل پایتون است، به دو روش می‌توانید آن را دانلود کنید:

### روش اول — دانلود کل پروژه

از بالای صفحه روی دکمه سبز رنگ **Code** کلیک کرده و گزینه **Download ZIP** را انتخاب کنید.

سپس فایل را از حالت فشرده خارج نمایید.

### روش دوم — دانلود مستقیم فایل

روی فایل پایتون پروژه کلیک کرده و سپس گزینه **Download raw file** را بزنید تا فایل مستقیم دانلود شود.

## 🧩 مرحله دوم: نصب پایتون

اگر Python را روی سیستم خود ندارید:

1. وارد سایت رسمی Python شوید.
2. آخرین نسخه را دانلود و نصب کنید.
3. هنگام نصب، حتماً گزینه زیر را فعال نمایید:

```text
Add Python to PATH
```

---

## 📂 مرحله سوم: آماده‌سازی پوشه پروژه

1. فایل پایتون دانلود شده را داخل یک پوشه جدید قرار دهید.
2. فایل مدل سه‌بعدی خود را با نام:

```text

```

داخل همان پوشه کپی کنید.

3. فایل GIF متحرک خود را نیز با نام:

```text
gif.gif
```

داخل همان پوشه قرار دهید.
