# 📝 درباره پروژه

این پروژه یک سیستم پردازش تصویر و رندرینگ سه‌بعدی پیشرفته است که حرکت سر کاربر را به صورت زنده (Real-time) از طریق وب‌کم دریافت کرده و دقیقاً همان چرخش‌ها و زاویه‌ها را روی استخوان‌بندی (Bone Structure) یک مدل سه‌بعدی با فرمت `.glb` پیاده‌سازی می‌کند. 

این نرم‌افزار ترکیبی هوشمندانه از فریمورک هوش مصنوعی **MediaPipe** (برای استخراج نقشه سه‌بعدی صورت) و موتور قدرتمند **Panda3D** (برای رندرینگ گرافیکی و فیلترهای PBR) است.

---

# ⚡ ویژگی‌های فنی پروژه

* **کالیبراسیون خودکار (Auto Calibration):** در ۲ ثانیه اول اجرای برنامه (۶0 فریم اول)، زاویه نشستن کاربر نسبت به وب‌کم آنالیز و صفر می‌شود.
* **فیلتر حرکتی اسموث (Motion Smoothing):** جلوگیری از لرزش‌های ریز تصویر وب‌کم و شبیه‌سازی حرکت کاملاً نرم و طبیعی گردن انسان با نرخ محاسباتی داینامیک.
* **محدودیت‌های زاویه انسانی (Clamping):** محدود کردن زوایای چرخش سر (Yaw, Pitch, Roll) برای جلوگیری از چرخش‌های غیرطبیعی و دفرمه شدن استخوان کاراکتر.
* **پشتیبانی از رندرینگ مدرن:** استفاده از کتابخانه `simplepbr` برای رندر متریال‌های واقع‌گرایانه (PBR) روی مدل‌های سه‌بعدیِ GLB.
* **پس‌زمینه متحرک (Dynamic GIF Background):** قابلیت استخراج فریم‌های یک فایل GIF و رندر کردن آن به عنوان پس‌زمینه زنده در فضای سه‌بعدی.

# 📁 فایل‌های مورد نیاز (که خودتان باید تهیه کنید)

این مخزن شامل **تنها یک فایل اصلی پایتون** است. برای اجرای موفقیت‌آمیز برنامه، شما باید ۲ فایل زیر را شخصاً تهیه کرده و **دقیقاً در کنار فایل پایتون** قرار دهید:

1. **`destructor_head_model.glb`**: مدل سه‌بعدی سر یا کاراکتر شما با فرمت GLB (که دارای استخوان‌بندی سر با نام‌هایی مثل `mixamorig:Head` یا `Head` باشد).
2. **`gif.gif`**: یک فایل تصویر متحرک (GIF) دلخواه که به عنوان بک‌گراند کاراکتر نمایش داده می‌شود.

# 🚀 راهنمای جامع نصب و راه‌اندازی (قدم به قدم)

### مرحله اول: نصب پایتون
اگر پایتون را ندارید، آن را از [python.org](https://www.python.org) دانلود کنید. حتماً تیک گزینه **`Add Python to PATH`** را در مرحله اول نصب بزنید.

### مرحله دوم: آماده‌سازی پوشه پروژه
1. فایل اصلی پایتون این پروژه را دانلود کنید.
2. مدل سه‌بعدی خود را با نام `destructor_head_model.glb` و گیف خود را با نام `gif.gif` در همان پوشه‌ای که فایل پایتون قرار دارد کپی کنید.

### مرحله سوم: نصب کتابخانه‌ها
یک ترمینال (یا CMD) در پوشه پروژه باز کنید و دستور زیر را برای نصب پکیج‌های مورد نیاز اجرا کنید:

```bash
pip install opencv-python mediapipe numpy pillow panda3d simplepbr
